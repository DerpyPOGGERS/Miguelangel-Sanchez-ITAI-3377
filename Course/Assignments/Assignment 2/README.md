The problem was how to monitor pedestrian and traffic activity in a city without violating people’s privacy. The goal was to create a smarter system that can still track movement and traffic patterns. To solve this, the system used edge computing with a device called NVIDIA Jetson TX2. This system helps reduce data usage and protects peoples privacy. The system was able to detect and track thousands of objects per day and reached about 69% accuracy with speeds up to 20 frames per second.
