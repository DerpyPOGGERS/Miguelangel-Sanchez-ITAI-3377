The problem was understanding how devices in an IIoT system communicate with each other using different protocols. To solve this, I used simulation scripts written in Python to model how communication happens between sensors. Protocols that can be used were MQTT, CoAP, and OPC UA. The results showed that different protocols are designed for different situations. MQTT is efficient for many devices communicating at once, CoAP works well for low-power devices, and OPC UA is more secure for industrial systems.
