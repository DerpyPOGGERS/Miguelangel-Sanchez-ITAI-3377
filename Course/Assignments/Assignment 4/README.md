The problem was predicting future sensor data in an IIoT system. We used past data to forecast future values like using old temperature readings. The methods we found and used was removing outliers and normalizing the data, then splitting into training sets and tests. We used metrics like MAE and RMSE and TimeGPT to learn patterns. The results showed that the model was able to predict trends in the sensor data fairly well. Which means forecasting on monitor systems can detect issues before they happen.
